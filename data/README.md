# Data

The raw data files are not included in this repository due to file size.

This project used RNA stability / half-life measurements from Tani et al. and transcript sequence files from human RNA FASTA records.

Main data sources included:

- Tani et al. RNA stability supplemental tables
- Human RNA FASTA sequence files
- Expression, sequence, and RNA structure-derived features generated during the project

To reproduce the analysis, place the required input files in this folder or in a local `data/raw/` folder. The `data/raw/` folder is ignored by Git and should not be uploaded to GitHub.
